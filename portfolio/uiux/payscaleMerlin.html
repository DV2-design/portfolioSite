Payscale Merlin

Insight is a SaaS web application developed by PayScale to provide businesses with detailed data, analytics, and visualizations for job market metrics.

The application dashboard presents configurable data points and charts.

Key Project Skills/Components

  • UI/UX
  • Graphic Design
  • HTML / CSS
  • React.js
  • Node.js
  • Sencha ES6
  • Agile development
  • JIRA
  • git / bitBucket
  • WebStorm

My role with Insight was both as a developer and designer. I collaborated with PayScale’s internal design team on visual language development and component styles. We improved the various charts and graphs by emphasizing the relationship between data points and maintaining a clear information hierarchy. I then took final designs and crafted the HTML, CSS, and JavaScript for majority of the product.
